TO INSTALL BEARDSTAT WEB UI:

1) UNZIP DIRECTORY AND UPLOAD TO YOUR WEB SERVER (MUST SUPPORT PHP)

1b) SET PERMISSIONS AS NEED FOR DIRECTORY TO ALLOW WRITE ACCESS FROM WEB SERVER

2) NAVIGATE TO installer.php IN YOUR BROWSER
e.g. if you placed stats in the folder stats in the root web folder, goto
http://herpaderp.com/stats/installer.php

3) FOLLOW SETUP INSTRUCTIONS TO CONFIGURE DATABASE ACCESS 
(HINT: USE SAME VALUES FROM BeardStat config)

4) ENJOY
