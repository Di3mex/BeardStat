Player Statistics for the modern bukkit

BeardStat provides simple, yet powerful recording of player statistics (blocks mined, creepers killed, deaths by drowning, play time, last login, how much xp you've collected.).

Recent new additions to BeardStat include shearing, dyeing and crafting stats (crafting stats are broken due to a bukkit bug, will update as we find which build it's fixed in :) ), alongside improvements to make it easier and better.

BeardStat provides MySQL and flatfile (yml) as storage mechanisms for statistics.

Available since Grapefruit, we provide a web API, allowing you to integrate BeardStat into your website, with detailed stats breakdowns, top tens

BeardStat is also easy to work. Plugins can add their own custom stat collectors (such as breaking a block with a specific tool, or trying shear a creeper). BeardAch utilises BeardStat's API to provide statistic based achievements, alongside many other triggers.

Plugin

  • /laston [player] command to view when a player was last online
  • configure /laston to display all players (WARNING: SMALL WHITELISTED ONLY!, 20k+ players is a very bad idea to spam a player with)

Activate displaying all players via adding this to the config stats: lastonall:true
