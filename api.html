Player Statistics for the modern bukkit

Integration with BeardStat is a simple four step process.

  1. Add BeardStat as a soft dependency to your plugin.
  2. Use the code below to get the Statistics Manager
  3. Consult the documents as you work with the manager and other classes
  4. Build awesome statistics for your plugin!

The following code will get the stat manager from the plugin.
When you wish to use the StatManager, perform a if(stats!=null)
first to ensure it has been loaded. 


PlayerStatManager stats = null;
BeardStat bs = (BeardStat) Bukkit.getServer().getPluginManager().getPlugin("BeardStat");
        if(bs!=null && bs.isEnabled()){
            stats = bs.getStatManager();
        }

API Information

This section contains details about classes you will use when interacting with BeardStat.
Specifically methods that are considered stable/nessecary to use the API are discussed here.
Use of any other methods in the supplied classes is not reccomended, as they are liable to change,
or be dangerous. Also, the stat manager is not thread-safe, conduct all operations manipulating stats
inside the main thread.

BeardStat

Primarily a utility class for you, only one of the methods is of concern.
PlayerStatManager getStatManager() :: gives you the PlayerStatManager object

PlayerStatManager

The main class you will use to get stats on players
PlayerStatBlob getPlayerBlob(String name) :: Gets the PlayerStatBlob of a player, creating one if one does not exist
PlayerStatBlob findPlayerBlob(String name) :: Gets the PlayerStatBlob of a player, returns null if no records exist for that player

PlayerStatBlob

This class holds all the stats related to a single player.
PlayerStat getStat(java.lang.String cat,java.lang.String name)
:: returns the PlayerStat object for that stat
boolean hasStat(java.lang.String cat,java.lang.String name)
:: returns true if the player has a record for that stat

PlayerStat

Provides a nice layer around the raw stat.
int getValue() :: return current value of this stat
void setValue(int value) :: set value of this stat
void incrementStat(int i) :: add i to current value of stat
void decrementStat(int i) :: subtract i from current value of stat
